## Сборка
`npm i` - установка модулей  
`npm run dev` - dev режим сборки, поднимает локальный сервер и вотчит файлы  
`npm run start` - сборка проекта и запуск на 3000 порту
`npm run build` - сборка проекта  
`npm run test` - запуск тестов
